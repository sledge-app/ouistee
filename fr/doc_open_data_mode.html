Ouvrir en mode données.

Permet d’ouvrir la collection directement dans le mode données. Permet de gagner du temps si la structure de votre collection est terminée.

Sélectionner la Collection depuis le tableau de bord. Dans le panneau des propriétés, cocher la case « Ouvrir en mode données ».
