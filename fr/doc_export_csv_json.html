Export CSV & JSON

Export CSV & JSON.

Uniquement avec OuisteePlus.

Salut

Les formats CSV et JSON permettent l’échange de données entre applications. Le CSV permet le transfert de données de type texte tabulaire (tableurs), alors que le JSON est très répandu sur le Web.

Salut

2) En bas à gauche, vous pouvez choisir le type d’export que vous voulez.

Choisissez CSV ou JSON.

3) En haut à gauche, dans « Exporter à partir de » : 

Vous pouvez définir si vous souhaitez utiliser la fiche courante, celle qui est active (en cours de visualisation) dans le mode données ou la sélection courante. Cette dernière peut être soit l’intégralité de votre Collection, soit le résultat d’une recherche en cours effectuée depuis la Vue par list.

4) Dans l’espace principale remplissez les champs : 

  • Nom du Fichier (obligatoire) 
  • C’est le nom du fichier. Par défaut, c’est le nom de la Collection. Saisissez monExport pour avoir monExport.pdf.
  • Emplacement où sera sauvé le fichier (obligatoire) 
  • Choisissez ici un emplacement où sauvegarder le fichier sur votre ordinateur.
  • Titre (seulement pour JSON)
  • Sera affiché sur la page de garde. Par défaut le nom de la Collection.
  • Sujet (seulement pour JSON)
  • Sera affiché sur la page de garde. Par défaut la description (celle qui est visible sur le Tableau de bord) de la Collection.

5) Vous êtes maintenant prêt à exporter. 

Cliquer sur « Enregistrer le CSV -ou JSON- sur le disque ». Après un temps de traitement qui dépend de la taille de la Collection, votre nouveau document apparait à l’emplacement choisi sur votre ordinateur.

C’est une façon simple et rapide, avec un minimum de configuration, de créer des fichier d’échanges entre applications.

Note | Par ce que le CSV est seulement pour des données de type texte tabulaire, les données des Élements suivants ne seront pas exportées : Pièces jointes, Photos, Liste. Les données de l’Elément texte perdront aussi leur mise en forme.

Note | Par ce que le JSON est seulement pour les données de type texte, les données des Élements suivants ne seront pas exportées: Pièces jointes, Photos.
