Parcourir les données

Parcourir les données.

Ouistee propose deux façons de naviguer dans vos données : La vue par Fiche (par défaut) et la vue par liste. 

Salut

1) Vue par fiche.

C'est le mode d'affichage par défaut des données quand vous entrer dans le mode "Données".

C'est ici que vous pouvez modifier toutes vos données (ajout, modification, suppression). Chaque fiche (ou enregistrement) est représenté par page, reprenant la structure défini dans le mode Édition. En haut de la fiche, vous avez à disposition certaines commandes :

  • Le button "Ajouter", pour créer une nouvelle fiche (Numéro 1 dans l'image 1).
  • Le button "Supprimer", pour effacer fiche (Numéro 2 dans l'image 1).
  • Un slider, pour naviguer rapidement dans vos données (Numéro dans l'image 1).
  • Deux flèches (droite et gauche) pour naviguer d'une fiche à l'autre. (Numéro 4 dans l'image 1).

Quand vous entrez dans le mode données, la fiche la plus ressente est toujours affichée par défaut

Image 1.


2) La vue par liste.

  • C'est une bonne façon de naviguer dans vos données, et d'avoir une vision globale rapide. Vous l'obtenez en cliquant sur "Vue par liste" (numéro 5, image1).

note | Une fois sélectionné, le bouton "Vue par liste" devient "Vue par Fiche", permettant de revenir à la vue par Fiche.

note | La liste s'ouvre toujours sur l'index sélectionné dans la vue par fiche.

note | Le bouton "Vue par Liste" n'est disponible qu'une fois au moin qu'une fiche est créée.

  • Dans la vue par liste, chaque ligne est une fiche. 
  • La vue par liste propose un affichage en une colonne de tout ou partie des données, une ligne représentant une fiche. Vous pouvez configurer les données qui s’affichent via le panneau de configuration en cliquant sur le bouton 1 (image 3).
