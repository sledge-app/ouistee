Structure

Structure.

A l'ouverture d'une Collection, Ouistee présente le mode "Édition". C'est ici que vous pouvez créer l'interface utilisateur de votre base de données. Ouistee s'occupe de créer automatiquement la structure adéquate des données. Aucune connaissance technique n'est requise.

Salut

Le mode Édition.

  • Le mode Édition est fait de trois grandes parties : la barre d'outils (1), la barre latérale et ces trois onglets (2), et l'espace de travail (3).
  • La barre latérale contient trois parties, la liste d'éléments (en haut, 4), la liste des propriétés (5) qui affiche les propriétés de l'élément sélectionné dans l'espace de travail et enfin les catégories en onglet 3 (non illustré ici).
  • En glissant déposant des Élements depuis la barre latérale, vous créez l'interface utilisateur de votre base de données.
  • Une fois un élément sélectionné dans l'espace de travail, vous avez accès à ses propriétés dans la barre latérale pour le modifier ou le paramétrer. La barre d'outils affiche aussi les options disponibles (la plupart du temps "Supprimer").
  • Une fois que vous avez terminé, cliquer sur "Terminer l'édition" dans la barre d'outil. Ouistee vous amène dans le mode "Données" dans lequel vous pouver commencer à entrer vos données personnelles.
