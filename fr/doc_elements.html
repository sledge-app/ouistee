Les Éléments

Les Éléments.

Un Élément définit un morceau d’interface utilisateur que vous pouvez déposer sur l’Espace de travail pour constituer l’interface, la structure, de votre Collection. 

Salut

1) Déposer un Élément de la liste de gauche sur l’Espace de travail.

Alors que vous avez sélectionné un élement dans la liste de gauche, les emplacements disponibles se matérialisent (en bleu). Vous pouvez ainsi déposer l’Élément en dessous de ceux déjà déposés, ou au contraire l’insérer entre deux Éléments existants. Une fois déposé (cliquer pour déoser), vous avez toujours la possibilité de le déplacer en le sélectionnant.

2) Les Préférences.

Dans cette partie ne seront traités que les Propriétés communes à tous les Éléments. 

Avant toute chose, toute actions sur un Élément suppose de l’avoir préalablement sélectionné : cliquez simplement dessus. Un Élément sélectionné s’entoure d’une bordure bleue pointillée. 

Dans l’espace des Propriétés (en bas à gauche) vous trouverez : 

  • Couleur
  • Choisissez parmi les choix proposés une couleur pour l’arrière plan de l’Élément. Toutes les parties constituants l’Élément (texte, boutons) s’adaptent à l’arrière plan. Vous sélectionnez donc plus une déclinaison de couleurs qu’un simple arrière plan. 
  • Transparence (alpha)
  • Glisser le curseur pour ajuster la transparence de l’arrière plan.
  • Hauteur
  • Glisser pour ajuster la hauteur de l’Élément.
  • Label (le cas échéant)
  • Modifier ici le label de l’Élément.
