Élément Attachement

Élément attachement.

Uniquement avec OuisteePlus.

Salut

L’Elément Attachement permet d’ajouter une image à vos Fiches. Le but est ici d’illustrer, non de stocker un fichier.


Salut

1) Une fois l’Élément déposé sur l’Espace de travail, passer dans le mode données (Terminer l’édition).

2) En haut à droite de l’Élément, vous trouvez deux boutons. Un clique dessus vous permettra d’explorer votre ordinateur pour trouver le fichier à incorporer : 

  • Attacher : Le fichier est importé dans la base de données de Ouistee. L’inconvénient est la prise de poids potentiellement importante de la base de données. L’avantage est que le fichier sera inclus dans les Archives, donc totalement « portable » (il sera présent si vous ouvrez l’archive sur un autre ordinateur). Il fait parti intégrante de votre Collection.
  • Lier : Dans ce cas Ouistee ne stock qu’un lien vers le fichier qui se trouve sur votre disque dure. L’avantage bien sur est d’économiser la taille de la base de données de Ouistee. L’inconvénient est qu’il ne sera pas inclus dans les Archives. Ouvert sur un autre ordinateur, le lien vers le fichier sera cassé, à moins que le même fichier se trouve au même emplacement sur l’autre ordinateur. 

Vous pouvez utiliser les deux modes en même temps sur la même Fiche. Une fois importés ou liés, chaque fichier est représenté par une ligne contenant sur nom. Le lien est matérialisé par une icône « lien » (une chaine) à côté du nom. 

À droite se trouve les boutons d’actions : 

  • Pour le lien, un « œil » permet d’ouvrir le Finder à l’emplacement du fichier lié.
  • Pour un fichier importé, le bouton d’accès vous permet de récupérer le fichier à l’emplacement de votre choix.

À gauche de chaque ligne, une petite croix permet de supprimer le fichier, ou simplement le lien, le cas échéant.
