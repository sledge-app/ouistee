Export PDF

Export PDF.

Uniquement avec OuisteePlus.

Salut

Le format PDF est un excellent moyen de partager vos Collections avec vos connaissances. Vous pouvez envoyer ce fichier par email ou via tout type de réseaux sociaux. Ouistee propose d’exporter tout ou parti de vos Collections.

Salut

1) Aller dans le « Mode donnée » et cliquer sur le bouton Exporter dans la barre de menu.

Vous vous retrouvez dans le panneau d’export.

2) En bas à gauche, vous pouvez choisir le type d’export que vous voulez.

Dans notre cas choisissez « pdf » (si ce n’est pas déjà sélectionné)

3) En haut à gauche, regarder « Exporter à partir de : ».

Vous pouvez définir ici si vous souhaitez utiliser la fiche courante, celle qui est active (en cours de visualisation) dans le mode données ou la sélection courante. Cette dernière peut être soit l’intégralité de votre Collection, soit le résultat d’une recherche en cours effectuée depuis la Vue par list.

4) Dans l’espace principale remplissez les champs : 

  • Nom du Fichier (obligatoire) 
  • C’est le nom du fichier. Par défaut, c’est le nom de la Collection. Saisissez monExport pour avoir monExport.pdf.
  • Emplacement où sera sauvé le fichier (obligatoire) 
  • Choisissez ici un emplacement où sauvegarder le fichier sur votre ordinateur.
  • Auteur
  • Sera affiché sur la page de garde. 
  • Titre
  • Sera affiché sur la page de garde. Par défaut le nom de la Collection.
  • Sujet
  • Sera affiché sur la page de garde. Par défaut la description (celle qui est visible sur le Tableau de bord) de la Collection.
  • Ajouter un page de garde
  • Ajoute une page de garde avec le titre, l’auteur et le sujet.

5) Vous êtes maintenant prêt à exporter.

Cliquer sur « Enregistrer le PDF sur le disque ». Après un temps de traitement qui dépend de la taille de la Collection, votre nouveau document PDF apparait à l’emplacement choisi sur votre ordinateur.

C’est une façon simple et rapide, avec un minimum de configuration, de partager vos données.
