Import CSV

Import CSV.

Uniquement avec OuisteePlus.

Salut

Le format CSV est très répendu dans les logiciels permettants de gérer des données de type tabulaire (Tableur comme Excel ou Number). Ouistee est capable de récupérer ce format pour créer une nouvelle Collection, avec sa structure et ses données.

Salut

1) Depuis le Tableau de bord, cliquer sur « Importer ».

Vous obtenez un premier écran sur lequel vous pouvez choisir le séparateur de votre fichier CSV (par défaut un point virgule). En cliquant sur « Merci de sélectionner le fichier » vous êtes invité à trouver votre fichier CSV sur votre ordinateur. Cliquer alors sur « Ouvrir ».

2) Vous arrivez sur un écran vous donnant accès à quelques réglages :

  • Cocher la case « La première ligne désigne les labels » si l’entête de votre CSV contient la description des colonnes.
  • La tableau propose ensuite une simulation de ce que donnera l’interprétation du fichier, en prenant comme exemple la première ligne de données (ou la deuxième si la première est occupée par les labels). Chaque ligne du tableau représente un Elément qui sera créé, avec son label (le cas échéant), sa valeur, et son type. Ce dernier peut être modifié dans le menu déroulant qui propose une liste d’Éléments gérants des valeurs de type texte (Texte, Texte avec label, email...).

3) Enfin le bouton « Créer la Collection » vous ramène (après un temps de traitement dépendant de la taille du CSV) au tableau de bord, sur lequel vous trouvez la nouvelle Collection fraichement créée.
