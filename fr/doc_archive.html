Archiver

Archiver.

Ouistee met à disposition un outil pour archiver votre base de données dans un unique fichier .ee, qui contient toute la base et ses documents liés (images, documents…). Vous pouvez alors le garder sur un support de sauvegarde, le transférer sur un autre ordinateur…

Salut

1) Créer une archive.

- Aller au menu Archive > Créer une Archive

Vous êtes invité à sélectionner un emplacement pour votre nouvelle archive, puis cliquer sur "Ouvrir".

Vous obtenez alors un fichier .ee que vous pouver garder pour sauvegarde ou pour le transporter.

2) Restaurer une archive

- Aller au menu Archive > Restaurer à partir d'une archive

- Ouistee vous informe que la restauration supprimera les précédentes données.. Cliquer sur "ouvrir" pour continuer.

- Vous êtes invité à localiser l'archive .ee sur votre ordinateur. Cliquer alors sur "Ouvrir".

- Une fois complété, la restauration aura remplacée toute les anciennes données par celle de l'archive. L'archive n'est pas modifiée par le l'opération et peut être réutilisée à loisir.
