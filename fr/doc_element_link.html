Élément Liens

Élément Lien.

Permet d’ajouter des hyperliens à vos Fiches.

Salut

Une fois que l’Elément a été déposé dans l’espace de travail, aller dans le mode Données (« Terminer l’édition » dans la barre des menus).

  • 1) Pour ajouter ou éditer un lien, cliquer sur le petit crayon à droite de l’Elément.
  • 2) Un champ de texte éditable apparait, vous pouvez saisir votre lien.
  • 3) Une fois fait, cliquer sur la petite coche sur la droite pour valider la saisie.
  • 4) Le lien devient actif (cliquer pour ouvrir un navigateur Web). Si il est disponible, le favicon du site concerné s’affiche.
